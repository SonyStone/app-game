# Grease Pencil TypeGPU

Standalone Solid + TypeGPU grease-pencil sketch app.

## Scripts

```bash
pnpm --filter @app-game/grease-pencil-typegpu dev
pnpm --filter @app-game/grease-pencil-typegpu build
```

The standalone dev server runs on `http://localhost:3060/`.

The app is also mounted in `@app-game/web` at `/grease-pencil-typegpu`.
